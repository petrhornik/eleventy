Eleventy

Eleventy is simple blazing fast static website generator. It allows creating websites from templates written for example in markdown or html.

Website types

There are two different types of websites, that is static and dynamic.

Static websites

Static websites are served as they are on disk without (any) content changes. They are fast, reliable, small.

Dynamic websites

On other hand dynamic pages allows realtime interaction, dynamic forms, personalization.
